# Pipeline System Optimization

Class assignment for Process Optimization course.

Two-variable nonlinear optimization of pipe diameter and pump efficiency to minimize total system cost (pipe + energy + pump).

## Demo

[Live App](https://pipeline-optimization-jzzpxeq2nalj95sq9bfaff.streamlit.app/)

## Stack

Python, Streamlit, NumPy, SciPy, Matplotlib

## Run locally
```bash
pip install streamlit numpy scipy matplotlib
streamlit run app.py
```
